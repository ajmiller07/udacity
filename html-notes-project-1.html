HTML

HyperText Markup Language documents create the bulk of what we know as the web. These documents consist of content, markup, refences to other documents, and links to other pages. The markup aspect of this, how you make the content look, is largely manipulated through the use of tags. Tags define the elements (the containers of content and building blocks of structure) and tell the browser what and how to display content and structure. Tags are not visible in the browser.

Tag Structure and Types

Inline vs Block:

Inline: Contents of the tag are displayed inline with text and other elements. For instance a picture element tag will dipslay in the middle of, or wrap inline with text unless nested in a block tag. Examples of inline tags covered in the first lesson include:

  • -tags for pictures <img>
  • -line breaks <br>
  • -links <a href=___>
  • -the container element span <span class=___>
  • -bold <b>
  • -italicized <em>

Block: Contents of block tags are not wrapped inline with text and occupy there own virtual space, to which parameters (like height x width) can be set. Examples of block tags covered in the first lesson include:

  • -tags for paragraphs <p>
  • -the container element div <div class=___>

Closed vs Void:

Closed: Closed tag elements begin with an opening tag (<some_tag>) and must be closed by a corresponding closing tag to surround the content for which it is intented to markup. A closing tag includes a forward slash prior to the tag name (</some_tag>). Examples of closed tag elements covered in the first lesson include:

  • -links <a href=___>
  • -the container element span <span class=___>
  • -bold <b>
  • -italicized <em>
  • -tags for paragraphs <p>
  • -the container element div <div class=___>

Void: Void tag elements contain no content outside of there attributes and therefore do not need a closing tag. Examples of closed tag elements covered in the first lesson include:

  • -tags for pictures <img>
  • -line breaks <br>

Attributes:

Attributes define and describe characteristics of tags. Where or what they refer or link to, what CSS class the container belongs to, shape and width of a container, etc. Examples of tags with attributes covered in the first lesson include:

  • -tags for pictures <img src="url" alt="text"> (attr= src= and alt=)
  • -the container element div <div class=___> (attr= class=)
  • -the container element span <span class=___> (attr= class=)
  • -links <a href="link_to_site_goes_here.com"> (attr= href= )

Basic Document Structure

Doctype:

Noted on the first line of the document, tells the browser or editor what type of HTML is being used. In HTML5 the doctype is denoted simply as HTML. The tag used to show doctype looks like this <!DOCTYPE HTML>

HTML Tag:

The <html> tag surrounds the entire content of the HTML document </html>

Head:

The head section, tagged by opening and closing <head> tags, contains metadata and information about the document including noting the use of CSS and javascript. The head section also can contain the title information tag <title> which the browser diplays on the title bar or tab.

Body and Content:

The body of the document, surrounded by opening and closing <body> tags, contains the content and structure of the page.
